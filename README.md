# Homebrew Tap for Just Speak to It

This is the official [Homebrew](https://brew.sh/) tap for [Just Speak to It](https://justspeaktoit.com), a native macOS voice transcription app.

## Installation

```bash
brew tap crmitchelmore/justspeaktoit
brew install --cask justspeaktoit
```

## Updating

```bash
brew upgrade --cask justspeaktoit
```

## Uninstalling

```bash
brew uninstall --cask justspeaktoit
brew untap crmitchelmore/justspeaktoit  # optional
```
